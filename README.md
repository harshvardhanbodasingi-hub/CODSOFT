
# CODSOFT – Python Internship Tasks

This repository contains the Python projects completed as part of the **CodSoft Python Programming Internship**.

## Task 1 – To-Do List Application

- Command-line to-do list manager
- Features:
  - Add, view, edit, delete tasks
  - Mark tasks as completed
  - Stores data in `tasks.json` using JSON
- File: `todo_list.py`

## Task 2 – Simple Calculator

- Menu-driven calculator
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with division-by-zero handling)
- File: `calculator.py`

## Task 3 – Password Generator

- Generates strong random passwords
- User can choose:
  - Length of password
  - Whether to include uppercase letters, digits, and symbols
- Uses Python `random` and `string` modules
- File: `password_generator.py`

## Task 4 – Rock-Paper-Scissors Game

- User vs Computer game in the terminal
- Random computer moves
- Keeps track of user and computer scores
- File: `rock_paper_scissors.py`

## Task 5 – Contact Book (Optional)

- Simple contact management system
- Features:
  - Add, view, search, edit, delete contacts
  - Stores data in `contacts.json`
- File: `contact_book.py`

---

## How to Run

1. Make sure Python is installed (`python --version`).
2. Download or clone this repository.
3. Run any task from the terminal:

```bash
python todo_list.py
python calculator.py
python password_generator.py
python rock_paper_scissors.py
python contact_book.py
